## The Multiverse of Chit-Chatbots 🚀✨

An interactive, multi-personality AI chatbot application built using **Streamlit** and powered by Google's **Gemini 2.5 Flash** model. This app allows users to switch between completely different chaotic, funny, and dramatic characters in real-time.

---

# 🎭 Features
* **Multiple Personalities:** Chat with popular characters like *Shinchan*, *Doraemon*, *Virat Kohli Fan*, *Ronaldo Fan*, *BTS Army*, *Gossip Girl*, and a *Topper Student*.
* **Dynamic Roleplay:** Powered by zero-shot prompt engineering to ensure characters never break character.
* **Clean & Responsive UI:** Built completely on Streamlit with interactive components and real-time status spinners.

---

# 🛠️ Tech Stack
* **Frontend:** Streamlit (Python-based framework)
* **LLM Core API:** Google GenAI SDK (`gemini-2.5-flash`)
* **Environment Management:** `python-dotenv`

---

# 🚀 How to Run the Project Locally

**1. Clone the Repository**

**2. Install Dependencies**
Make sure you have Python installed, then run:
```pip install streamlit google-genai python-dotenv```

**3. Setup API Key**
Create a .env file in the root directory of the project and add your Gemini API Key:
```GEMINI_API_KEY=your_actual_api_key_here```

**4. Run the Application**
Execute the following command in your terminal:
```python -m streamlit run app.py```
